# PAPER_REPRO_FLOW

## Goal
Produce a researcher-grade, reproducible dossier for a target paper.

## Intake
Collect:
1. arXiv ID/version
2. target outputs (equations/figures/theorems)
3. tolerance for mismatch
4. depth target (core or full dossier)

## Execution stages
1. Contribution map + problem framing
2. Assumption inventory (explicit + inferred)
3. Derivation trail with intermediate steps
4. Computational reproduction (Python/Julia)
5. Figure/table regeneration + paper comparison
6. Sensitivity and edge-case stress tests
7. Mismatch/failure analysis
8. Optional extension layer (clearly tagged)
9. Reproducibility appendix (environment, versions, seeds)

## Provenance requirements
Tag all key claims as:
- REPRODUCED_FROM_SOURCE
- INFERRED_ASSUMPTION
- NEW_EXTENSION
