# PLOTTING_POLICY

Required plot families:
- line sweeps
- heatmaps
- contours
- phase-region maps (if relevant)
- sensitivity plots
- asymptotic-check plots

Rules:
- SVG by default.
- Axis labels include units.
- Caption-ready takeaway required.
- Link each plot to run metadata.
- Fail quality gate if style/label/provenance checks fail.
