# Workflow Modes

## 1) Physics solve mode
- derive-first answer path
- mandatory checks
- multi-path verification for high-impact claims
- final interpretation

## 2) Paper reproduction mode
Intake first:
- arXiv id/version
- target equations/figures/theorems
- tolerance for mismatch
- depth target (core/full)

Then execute staged reproduction with provenance tags.

## 3) Parameter-space mode
- define constraints/ranges
- coarse-to-fine scans
- sensitivity and boundary analysis
- output SVG plots with metadata

## 4) Publishable draft mode
- promote only validated claims
- keep evidence linkage
- keep LyX track separation
