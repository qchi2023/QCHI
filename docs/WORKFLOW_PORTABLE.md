# WORKFLOW_PORTABLE

## Execution Flow
1. Intake task and clarify target output.
2. Identify physical regime and assumptions.
3. Build derivation path with explicit steps.
4. Run checks (units/limits/asymptotics/consistency).
5. Add numeric verification if applicable.
6. Generate artifact outputs (notes, plots, draft).
7. Tag claim provenance.
8. Produce reproducibility metadata.

## Paper Reproduction Mode
Ask first:
- arXiv ID/version
- target equations/figures/theorems
- acceptable numeric tolerance
- depth (core vs full dossier)

Then run staged reproduction and log all decisions.

## Deliverable Modes
- Quick: concise derivation + checks + conclusion
- Full dossier: complete derivation, plots, sensitivity, reproducibility appendix
